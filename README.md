# OCRBC

Happy Camper is a cross platform mobile application created as a better alternative to existing options available today for camping reservations in British Columbia.

## Installation

## Server

```bash
yarn install
```

## Client

```bash
yarn install
cd ios
pod install
```

## Run - (./client)

```bash
yarn ios
```

## Run - (./server)

```bash
yarn dev
```
